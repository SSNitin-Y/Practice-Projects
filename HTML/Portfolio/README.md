# HTML Project Checklist: Personal Portfolio Webpage

Build a single-page **Personal Portfolio Webpage** using only HTML.

Your goal is to create a webpage that introduces you, shows your skills and projects, and includes a way for someone to contact you.

---

## Project File

- [ ] Create a file named `index.html`

---

## 1. Page Setup

- [x] Add the basic HTML structure
- [x] Add `<!DOCTYPE html>`
- [x] Add the `<html>` tag
- [x] Add the `<head>` tag
- [x] Add the `<title>` tag
- [x] Add the `<body>` tag
- [x] Set the browser tab title to `Nitin | Portfolio`

Example:

```html
<title>Nitin | Portfolio</title>
```

---

## 2. Header Section

At the top of the page, include:

- [x] Your name as the main heading
- [x] A short tagline
- [x] A navigation menu

Suggested tagline:

```text
Aspiring Data Analyst and Tech Enthusiast
```

Navigation links should go to:

- [x] About Me
- [x] Skills
- [x] Projects
- [x] Contact

The links should move to sections on the same page.

Example:

```html
<a href="#skills">Skills</a>
```

---

## 3. About Me Section

Create a section titled **About Me**.

Include:

- [x] A heading: `About Me`
- [x] One paragraph introducing yourself
- [x] A profile image

Image options:

- [x] Use an online placeholder image
- [x] Or use an image saved in your project folder

---

## 4. Skills Section

Create a section titled **Skills**.

Use an unordered list.

Include at least 5 skills:

- [ ] HTML
- [ ] Python
- [ ] SQL
- [ ] Power BI
- [ ] Data Analysis

HTML element to practice:

```html
<ul>
  <li>HTML</li>
  <li>Python</li>
</ul>
```

---

## 5. Projects Section

Create a section titled **Projects**.

Add information about **two projects**.

Each project should include:

- [ ] Project name as a subheading
- [ ] Short description
- [ ] Technologies used
- [ ] A link labeled `View Project`

Suggested projects:

- [ ] TempsBot
- [ ] Connecticut Housing Affordability Analysis

The project links do not need to work yet. You can use `#` as the link destination.

Example:

```html
<a href="#">View Project</a>
```

---

## 6. Education Section

Create a section titled **Education**.

Include:

- [ ] University name
- [ ] Degree
- [ ] Graduation year

Practice using either:

- [ ] An ordered list
- [ ] Or a table

---

## 7. Contact Section

Create a section titled **Contact Me**.

Add a simple contact form with:

- [ ] Name input
- [ ] Email input
- [ ] Message box
- [ ] Submit button

Also include:

- [ ] LinkedIn link
- [ ] GitHub link

The form does not need to submit information anywhere yet.

---

## 8. Footer

At the bottom of the page, include:

- [ ] A footer section
- [ ] Copyright text

Use this text:

```text
© 2026 Nitin. All rights reserved.
```

---

---

## Extra Challenge

After completing the basic version, add:

- [ ] A link in the footer that takes the user back to the top of the page
- [ ] A downloadable resume link using an `<a>` tag
- [ ] An image for each project
- [ ] A horizontal line between major sections using `<hr>`

---

## Suggested First Step

Start by creating only these parts first:

- [ ] Basic HTML structure
- [ ] Header
- [ ] Navigation links

After that, paste your code for review before moving to the next sections.
